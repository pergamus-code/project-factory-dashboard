GRAFANA CONNECTION MANUAL: THE IT ADMIN GUIDE

A step-by-step guide to bridging a factory database to Grafana for live, real-time TV dashboards.

🏗️ PART 1: THE MENTAL MODEL (HOW GRAFANA WORKS)

Grafana is a Web Server Application.
It runs invisibly in the background of the host machine.

[Your Browser] 🌐 -----> [localhost:3000] -----> [Grafana Service] -----> [MySQL Database]

  • localhost = The machine you are currently sitting at.
  • :3000 = The specific port Grafana uses to broadcast its user interface.

🛡️ PART 2: THE PRE-FLIGHT CHECK (THE READ-ONLY ACCOUNT)

Before you ever open Grafana, you must secure the database. You will never use the root/admin password.

Why?

Grafana executes raw SQL queries. If an Admin account is used, a typo in a dashboard could delete the entire factory database.

The Setup (Done by the DBA inside MySQL):

  1. Create a "Service Account" (e.g., grafana_reader).
  2. Assign a strong password.
  3. Grant SELECT permissions ONLY.
[Admin Account] 👑 ---> Can SELECT, INSERT, UPDATE, DROP (Blocked from Grafana)
[Reader Account] 👓 ---> Can ONLY SELECT (Connected to Grafana)

🔌 PART 3: MAKING THE CONNECTION ACTIVE

You are sitting at the factory server PC (or testing on your Mac). Grafana is installed and running in the background.

Step 1: Open the Portal

  • Open Chrome, Safari, or Edge.
  • Type http://localhost:3000 into the address bar.
  • Log in (Default is usually admin / admin on the very first launch).

Step 2: Add the Data Source

  • Look at the left-hand sidebar.
  • Go to ⚙️ Configuration > Data Sources.
  • Click the blue [ Add data source ] button.
  • Search for and select MySQL.

Step 3: Configure the Network Settings

You must tell Grafana exactly where the database lives and hand it the "Reader" keys. Fill out these fields:

  • Host: localhost:3306
    (Note: 3306 is the standard default port for MySQL. If the DB is on another server, it might look like 192.168.1.55:3306)
  • Database: factory_production_db
  • User: grafana_reader
  • Password: [The Reader Password]

Step 4: The Test

  • Scroll to the bottom of the page.
  • Click [ Save & test ].
  • Grafana will take the grafana_reader password, knock on the MySQL port (3306), and ask for entry.
  • ✅ If successful, a green banner appears: "Database Connection OK"

💻 PART 4: THE AFTERMATH (BUILDING THE VISUALS)

To build a chart, you must write the exact SQL query.

The Workflow:

  1. Create a New Dashboard.
  2. Add a New Panel (Chart).
  3. Select your MySQL Data Source.
  4. Type your clean SELECT statement:
SELECT
  production_time AS "time",
  machine_id AS "metric", 
  units_produced AS "value"
FROM 
  assembly_line_stats
WHERE 
  $__timeFilter(production_time)
ORDER BY 
  production_time ASC;
  1. Grafana sends this exact query to MySQL, MySQL does the math, and Grafana draws the line chart.
